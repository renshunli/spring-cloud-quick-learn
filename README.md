# spring-cloud-quick-learn
spring-cloud 主要的几个概念 简单的搭建 网关 服务 消费 注册中心 负载 高可用  
依次启动服务 eureka eureka2 server server2 consumer zuul  
eureka 注册中心  
eureka2 注册中心  
server 服务提供者  
server2 服务提供者  
consumer 服务消费者  
zuul 路由网关  
实现了两个注册中心高可用。两个服务负载。
